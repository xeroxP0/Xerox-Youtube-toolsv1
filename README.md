インストール方法

1. このページにある緑色の<>codeというボタンを押してDownload zipを押してダウンロード
2. ダウンロードしたZipを解凍する
3. chrome://extensions/にアクセス
4. 右上にあるデベロッパーモードを有効化する
5. パッケージ化されていない拡張機能を読み込むを押してさっき解凍したフォルダの中にあるXerox-Youtube-toolsv1というフォルダを選択

この拡張機能はyoutubeの機能追加と見すぎてしまうShortsを無効化して作業効率を上げるツールです
