# Xerox YouTube Tools (ベータ)

短時間で見入ってしまう YouTube Shorts を非表示にして、作業効率を上げるための機能を追加する Chrome 拡張機能です。まだベータ版のため、不具合や要望がある可能性があります。
機能一覧
・ショートを非表示・強制的に通常再生に移行
これによって長時間スクロールするのを防ぎます
・再生速度を固定・音楽は通常再生させる
これによってyoutubeを見る時間を短縮します。ただし音楽は通常の速度で再生されます（ベータ版のため正しく検知されない可能性があり）
・強制的にシアターモード・ループ再生
動画を見やすくします
・便利ボタンを追加
これによって時間指定のURLをコピーするのと動画をスクショする機能を追加します
・ホームのおすすめを音楽のみ表示
これによってホームから動画を選択して時間をつぶしてしまうのを防ぎます
・コメント、関連動画、ライブチャットの非表示
これによって見続けてしまうのを防ぎます

## インストール方法（手順）
1. このリポジトリの緑色の「Code」ボタンから「Download ZIP」を選んでダウンロードします。  
2. ダウンロードした ZIP を解凍します。  
3. Chrome で `chrome://extensions/` にアクセスします。  
4. 右上の「デベロッパーモード」をオンにします。  
5. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダ内の `Xerox-Youtube-toolsv1` フォルダを選択します。  
6. 拡張機能が追加されたら、YouTube を開いて Shorts が非表示になっているか確認してください。

---

## 使い方
- 拡張機能をインストールして有効化するだけで、右上に専用の設定メニューを開くボタンが出てくるので好みに設定したら終わりです  
- 必要に応じてブラウザを再読み込みしてください。

---

## 注意事項
- 本拡張機能はベータ版です。想定外の挙動や未対応のページがある可能性があります。  
- YouTube のレイアウト変更により動作しなくなる場合があります。問題が発生したら報告してください。
