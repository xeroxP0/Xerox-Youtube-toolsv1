Xerox YouTube Tools (ベータ)
YouTube の視聴を時間効率よくするための Chrome 拡張機能です。特に短時間で延々と見てしまいがちな Shorts を抑制し、必要な機能だけに集中できるように支援します（ベータ版）。

主な特徴
Shorts の非表示・通常再生へ強制移行
長時間スクロールしてしまうことを防ぎます。
再生速度の固定（音楽は通常再生）
動画の再生速度を固定して視聴時間を短縮します。音楽は通常速度で再生するように優先判定しますが、ベータのため誤判定する場合があります。
シアターモードの強制・ループ再生
視聴をしやすくします（好みで有効化してください）。
便利ボタンの追加
時間指定 URL のコピー
動画のスクリーンショット（保存・コピー）
ホームのおすすめを「音楽のみ」表示
ホーム画面から無駄に動画を選んで時間を使ってしまうのを防ぎます。
コメント、関連動画、ライブチャットの非表示
注意をそらす要素を減らします。
インストール手順
このリポジトリの「Code」→「Download ZIP」で ZIP をダウンロードします。
ダウンロードした ZIP を解凍します。
Chrome で chrome://extensions/ にアクセスします。
右上の「デベロッパーモード」をオンにします。
「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダ内の Xerox-Youtube-toolsv1 フォルダを選択します。
拡張機能が追加されたら YouTube を開き、設定を確認してください。
使い方
拡張機能をインストールして有効化するだけで動作します。
画面右上に表示される設定ボタンから各機能のオン／オフを切り替えられます。
変更後は必要に応じてブラウザを再読み込みしてください。
注意事項（重要）
本拡張機能はベータ版です。想定外の挙動や未対応ページが存在する可能性があります。
YouTube のレイアウトや仕様変更により一時的に動作しなくなる場合があります。
音楽判定や一部の機能は誤検知する可能性があります。
